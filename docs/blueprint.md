# **App Name**: Lucky Bingo

## Core Features:

- Dynamic Bingo Card Generation: Programmatically generate unique 5x5 bingo cards (cartels) with randomized numbers within standard ranges for B, I, N, G, O columns.
- Interactive Cartel Selection: Allow users to browse and select up to five unique bingo cartels to play in the current round.
- Live Game Information Display: Show a real-time countdown timer for game rounds and dynamically update the total stake based on the number of selected cartels.
- Selected Cartel Detail View: Display a magnified view of all selected bingo cartels, showing their unique number configurations for easy review.
- User Account Dashboard: Present the user's unique ID and current game balance on the home screen, providing an overview of their account status.
- Game State & User Data Persistence: Store and retrieve user balances, selected cartels, and essential game state for seamless continuation across sessions.
- AI-Powered Bingo Call Tool: An AI tool generates engaging and descriptive commentary for each called bingo number, enhancing the game's atmosphere.

## Style Guidelines:

- Primary interactive color: A vibrant teal (#17C8B8), representing clarity and a forward-thinking digital aesthetic, used for key actions and highlights.
- Background color: A deeply muted dark teal (#161B1B), providing a sophisticated, immersive dark theme with subtle depth that enhances focus on game elements.
- Accent color: A lively bright green (#38E26E), utilized for successful states, confirmations, and elements requiring a positive, active emphasis, creating contrast with the primary teal.
- Urgency and warning color: A striking red (#EF4444), specifically for countdown timers, alerts, or critical information requiring immediate user attention.
- Headline and body text font: 'Space Grotesk' (sans-serif), chosen for its modern, slightly futuristic, and highly legible quality, fitting a digital gaming context.
- Iconography should utilize clean, line-based vector icons (from 'lucide-react' library), ensuring clarity, scalability, and a consistent modern interface aesthetic across all UI elements.
- Mobile-first, dark-themed responsive layout featuring clear, spacious content blocks and a fixed bottom navigation bar for intuitive game navigation. Key game elements are presented within central, defined areas.
- Subtle, fluid CSS transitions for button interactions, cartel selection, and page changes to provide smooth user feedback and enhance the dynamic feel of the game.
